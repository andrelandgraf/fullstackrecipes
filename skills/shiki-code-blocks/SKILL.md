# Shiki Code Blocks

To set up Shiki Code Blocks, refer to the fullstackrecipes MCP server resource:

**Resource URI:** `recipe://fullstackrecipes.com/shiki-code-blocks`

Use the fullstackrecipes MCP server to fetch this resource and follow the setup instructions.

If the MCP server is not set up, please use:

`curl https://fullstackrecipes.com/api/recipes/shiki-code-blocks`
