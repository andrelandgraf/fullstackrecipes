## Create a Next.js App

Start with a fresh Next.js application using the latest version.

### Initialize the project

```bash
bunx create-next-app@latest my-app
```

Recommended options to select during the setup:

- **TypeScript**: Yes
- **ESLint**: No
- **Tailwind CSS**: Yes
- **src/ directory**: Yes
- **App Router**: Yes
